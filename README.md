# UsbWriteProtectTray

A tray application for Windows to turn USB write protect on and off.

## Purpose

Using this tray application, you can toggle write protect on and off
for USB drives. The tray application signals current state in text
and with a red or green drive icon. This makes it easy to determine
the current state.

## What it does

USBWriteProtectTray changes the registry key *WriteProtect* in
*SYSTEM\CurrentControlSet\Control\StorageDevicePolicies*.

Using a tray icon is more convenient than using *regedit*.
